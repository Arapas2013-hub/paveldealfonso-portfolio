# paveldealfonso — Portfolio

## Instalación (3 pasos)

### 1. Instalar Node.js
Si no lo tienes, descárgalo en: https://nodejs.org (versión LTS)

### 2. Instalar dependencias
Abre la carpeta del proyecto en la terminal y ejecuta:
```
npm install
```

### 3. Arrancar
```
npm run dev
```

Abre tu navegador en: **http://localhost:3000**

## Tus fotos
Todas tus fotos están ya dentro de `/public/imgs/`:
- `eventos-01.png` a `eventos-17.png` → carrusel de eventos
- `exposiciones-01.png` a `exposiciones-04.png` → carrusel de exposiciones

Para añadir o cambiar fotos, simplemente reemplaza los archivos en esa carpeta y actualiza los nombres en `src/app/page.tsx` en los arrays `EVENTOS`, `EXPOSICIONES` y `EDICION`.
